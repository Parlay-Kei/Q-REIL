# Naming Registry

Canonical names for projects and systems. **No alternate spellings allowed.**

## Q REIL

| Field | Value |
|-------|--------|
| **Human name** | Q REIL |
| **Slug** | q-reil |
| **System key** | q_reil |
| **Repo name** | q-reil |
| **Vercel project name** | q-reil |
| **Base route** | /q-reil |

### Rules

- Use **Q REIL** in UI and documentation (human-facing).
- Use **q-reil** for URLs, repo, and Vercel project (slug).
- Use **q_reil** for env vars, DB keys, and code identifiers (system key).
- No alternate spellings (e.g. no "Q-Reil", "qreil", "QREIL" as product name).
